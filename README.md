## 🚀 เริ่มต้นใช้งาน (Getting Started)

ทำตามขั้นตอนด้านล่างนี้เพื่อรันโปรเจกต์บนเครื่องของคุณ:

### 1. คลอนรีโพสิทอรี (Clone the Repository)
\`\`\`bash
git clone https://github.com/ppilovee01/Git-Day1-Test
cd [repo-name]
\`\`\`

### 2. ติดตั้ง Dependencies
\`\`\`bash
npm install
# หรือ yarn install
\`\`\`

### 3. ตั้งค่าสภาพแวดล้อม (Environment Variables)
สร้างไฟล์ `.env` ไว้ที่ root folder แล้วใส่ค่าตามนี้:
\`\`\`env
DATABASE_URL=your_database_url
API_KEY=your_api_key
\`\`\`

### 4. รันโปรเจกต์ (Run the Project)
\`\`\`bash
npm run dev
\`\`\`
เปิดไปที่ `http://localhost:3000` เพื่อดูผลลัพธ์ 🥳

## 🤝 การมีส่วนร่วม (Contributing)

ยินดีรับฟังทุกคำแนะนำและโค้ดรีเควสครับ! หากต้องการสนับสนุน:
1. Fork โปรเจกต์นี้
2. สร้าง Feature Branch (`git checkout -b feature/AmazingFeature`)
3. Commit การเปลี่ยนแปลง (`git commit -m 'Add some AmazingFeature'`)
4. Push ไปที่ Branch (`git push origin feature/AmazingFeature`)
5. เปิด Pull Request
